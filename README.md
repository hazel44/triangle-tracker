# Project Name.
Triangle Tracker.
## Author.
Miriam Hazel Oduor.
## Description.
This application enables users to know whether a Triangle is Equilateral, Isosceles, Scalene
or not a triangle at all by inputing numerical values of the sides of the triangle.
